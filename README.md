# ✈️ Airline Booking System (C++)

## 📌 Overview

This is a simple airline booking system developed in C++.
The project was created as part of my learning journey to practice programming fundamentals.

---

## ✨ Features

* 👤 User login with password (3 attempts)
* 🎫 Three ticket types:

  * VIP (1000$)
  * Medium (600$)
  * Normal (400$)
* 💺 Book multiple seats
* 💰 Payment and change calculation
* 🔁 Repeat booking option

---

## 🛠️ Technologies

* C++
* iostream

---

## ▶️ How to Run

```bash
g++ main.cpp -o program
./program
```
